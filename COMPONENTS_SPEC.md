# Component Specifications

Implement the following modular components exactly as specified. All components must be fully responsive (Mobile-first, scaling flawlessly to desktop).

## 1. Navigation Bar (`Navbar`)
* **Context:** Sits transparently on top of the Dark Forest Green hero background.
* **Left Section:** App/Brand Logo (Squared icon with a stylized white 'G' on a lime/green background).
* **Center Section:** Links (`Case Studies`, `Design Artifacts`, `About`). Text color: White, medium weight, subtle opacity on hover.
* **Right Section:** "Let's Connect" button with a right arrow icon ($\rightarrow$), styled as a pill-shaped Primary Lime Button.

## 2. Hero Section (`Hero`)
* **Background:** Deep Forest Green.
* **Layout:** 2-column grid on desktop, 1-column stack on mobile.
* **Left Column:**
    * *Badge:* Small pill container with a wave emoji: "👋 Hi, I'm Goodness Ayano".
    * *Main Heading:* Large, impactful typeface. "Designing Products **People Love** and **Businesses** Value". Accentuate "People" and "Businesses" using the Vibrant Lime Green color.
    * *Subheading:* Short paragraph describing user-centered digital experiences in a clean, high-contrast white/muted text.
    * *CTAs:* Row containing "View Case Studies" (Primary Lime Button) and "Let's Connect" (Secondary Outline Button with arrow).
    * *Metrics Row:* Flat display showing:
        * "Products Designed: 15+"
        * "Experience: 4 Years+"
        * "Open to: Product Design Opportunities"
* **Right Column:**
    * Features a 3D animated Claymation-style avatar character wearing a cap and AirPods, set against a rounded rectangle light green background.
    * Flanked by abstract, multi-pointed **Electric Orange vector starbursts** in the bottom-left and top-right corners of the frame.

## 3. Social Proof/Trust Bar (`TrustBar`)
* **Background:** Pure White (`#FFFFFF`).
* **Layout:** Asymmetric horizontal split.
    * *Left side (40%):* Header "Trusted By Innovative Teams" + descriptive paragraph.
    * *Right side (60%):* A clean, horizontally scrolling or wrapping row of monochromatic/colored partner logos (educational institutions, tech teams, organizations).

## 4. Case Study Card (`CaseStudyCard`)
* **Layout:** White background, vertical stack.
* **Top:** Image container with a rounded aspect ratio showing the project interface mockup.
* **Middle:** * A small utility badge (e.g., "Mobile App", "SaaS Platform", "Community Platform") using a soft dark green or light gray background.
    * `Title` - Bold, clean black text.
    * `Description` - Short paragraph in muted gray text.
* **Bottom:** Two small inline text action links: "Read Case Study" and "View Live Site $\nearrow$".

## 5. Design Artifacts Grid (`ArtifactsGrid`)
* **Layout:** Grid system (4 columns on desktop, 2 on tablet, 1 on mobile).
* **Items:** Uniform square or slightly rectangular cards showcasing clean, high-fidelity interface screenshots, style guides, and design explorations.

## 6. Footer Contact Form (`ContactFooter`)
* **Background:** Light Off-White background transitioning to a Dark Forest Green base bar at the very bottom.
* **Layout:** Two columns.
    * *Left Column:* Title "Let's Create Better Product Experiences", descriptive prompt, a row of circular social media icons (X/Twitter, LinkedIn, Dribbble, Behance), and a repeating stylized geometric "G" wallpaper pattern block.
    * *Right Column:* A clean card containing form fields (`Name`, `Email Address`, `Message text area`) and a prominent "Send Message" action button.
* **Bottom Bar:** Deep Forest Green stripe with centered text: "© 2026 Goodness Ayano. All rights reserved."
